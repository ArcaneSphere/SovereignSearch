Help & Documentation

Welcome to the Orbis wallet help section. This guide will help you understand all the features and functionalities available in the application.

ğŸ“Š

Overview

The Overview page provides a quick snapshot of your wallet and network status at a glance.

Features:

  • Wallet Overview: Displays your wallet address and current DERO balance
  • Network Status: Shows whether you're connected to Mainnet or Testnet, along with the current block height
  • Total Transactions: Displays the total number of transactions in your wallet
  • Total Fees Spent: Shows the cumulative transaction fees you've paid
  • Mining Status: Quick view of your mining activity and current hashrate
  • Avatar System: Choose an NFA as your profile avatar
ğŸ’°

Wallet

The Wallet page gives you detailed information about your DERO balance and transactions.

Features:

  • Wallet Address: Your unique DERO address with one-click copy functionality
  • Current Balance: Real-time display of your DERO balance
  • Transaction Pie Chart: Visual breakdown of incoming, outgoing, and coinbase transactions
  • Transaction Filters: Filter by type (All, Incoming, Outgoing, Coinbase)
  • Full Scan: Complete blockchain rescan to ensure all transactions are detected

Send DERO & Tokens

Send DERO and tokens to other addresses with real-time validation.

Features:

  • Address Validation: Real-time verification of destination addresses
  • Name Resolution: Support for registered account names (e.g., @myname)
  • MAX Button: Quickly send your entire balance
  • Ring Size Selection: Choose privacy level (2-128, default 8)
  • Token Support: Send any custom token from your wallet
  • Balance Check: Validates if address exists on blockchain

Transactions

The Transactions page displays your complete transaction history with powerful filtering and export capabilities.

Features:

  • Transaction List: View all incoming, outgoing, and coinbase transactions with details
  • Search Function: Quickly find specific transactions by TXID
  • Type Filters: Filter by Incoming, Outgoing, or Coinbase
  • Pagination: Navigate through your transactions with customizable items per page
  • Refresh & Rescan: Update your transaction list or perform a full blockchain rescan
  • Export to JSON: Export all or filtered transactions for backup or analysis
  • Date Range Export: Export transactions within specific date ranges

Services

Access DERO blockchain services including integrated addresses and name registration.

Features:

  • Integrated Address Generator: Create addresses with embedded payment details
    • Destination Port (0-65535)
    • Amount in DERO (optional)
    • Comment/Message (max 256 chars)
  • Account Name Registration: Register human-readable names for your wallet
    • Minimum 6 characters
    • Real-time availability check
    • No spaces allowed
    • Smart Contract SCID: 0000...0001
  • My Account Names: Manage your registered names
    • View all names you own
    • Transfer ownership to another address
    • Real-time blockchain validation
    • Name resolution support

Mining

The Mining page allows you to mine DERO directly from your wallet using integrated mining functionality.

Features:

  • Start/Stop Mining: Toggle mining on and off with a single click
  • Thread Control: Adjust the number of CPU threads dedicated to mining
  • Mining Statistics: Real-time display of your hashrate, blocks found, and mining duration
  • Efficiency Metrics: Track hashes computed, miniblocks found, and total attempts
  • Session Tracking: Monitor your current mining session with automatic statistics updates

Assets (NFTs/NFAs)

The Assets page displays your non-fungible assets and collectibles stored on the DERO blockchain.

Features:

  • Asset Gallery: Visual display of all your NFAs and NFTs with thumbnails
  • Scan Assets: Automatically detect all assets by scanning the blockchain
  • Type Filtering: Filter assets by type (NFA, NFT, Others)
  • Asset Details: Click on any asset to view full details and metadata
  • NFT Display/Retrieve: Display NFTs on blockchain or retrieve them back
  • Send Assets: Transfer assets to other addresses
  • Set Avatar: Use NFAs as your profile avatar
  • Manual Addition: Add specific assets by entering their SCID
  • Import/Export: Backup your gallery using JSON files

Tokens

The Tokens page manages all your fungible tokens (custom tokens and currencies) on the DERO blockchain.

Features:

  • Token Gallery: Display all tokens you hold with their balances
  • Scan Tokens: Automatically discover all tokens in your wallet
  • Send Tokens: Transfer tokens to other addresses with validation
  • Balance Display: Real-time token balances with proper decimal formatting
  • Add Token: Manually add tokens by entering their SCID
  • Import/Export: Manage your token list with JSON backup files

Settings

Configure your wallet connection and preferences.

Features:

  • XSWD Address: Configure the address and port of your XSWD daemon (default: 127.0.0.1:44326)
  • Node Selection: Choose your preferred DERO node
  • Avatar Settings: Manage your profile avatar
  • Import/Export Settings: Backup and restore your configuration
  • Local Storage: Settings are stored locally in your browser

Getting Started

First Steps:

  1. Start XSWD: Make sure your XSWD daemon is running (dero-wallet-cli with --rpc-server flag)
  2. Configure Settings: Enter your XSWD address in Settings if different from default
  3. Connect Wallet: Click the "Connect Wallet" button in the top bar
  4. Explore: Navigate through the different sections to view your balance, transactions, assets, and more

Tips:

  • Always keep your XSWD daemon running while using Orbis
  • Use Export features to backup your transaction history and asset galleries
  • The Overview page provides quick access to all essential information
  • Mining can be resource-intensive - adjust threads according to your system
  • Use account names for easier transfers (e.g., send to "myname" instead of full address)
  • Integrated addresses are useful for merchants requiring specific payment details
